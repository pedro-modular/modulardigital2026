Para desenvolver estratégias de marketing digital é indispensável um serviço de Web Analytics. Permite analisar os dados de navegação que os usuários geram ao consultar o seu website, ajuda-o a perceber o que os usuários procuram e porque entram no seu website.

**Informações fornecidas por uma análise Web Analytics**

*   Número total de acessos, visitantes únicos,…
*   Duração das visitas
*   Origem dos visitantes (motores de pesquisa, visitas diretas, …)
*   Horas / dias / meses que lhe oferecem mais retorno em vendas online
*   Campanhas de publicidade online que lhe permitiram maior retorno
*   Páginas onde o visitante abandonou o website

**Objetivos de Web Analytics**

Este serviço tem como objetivo a interpretação e estudo dos dados, com os quais descobrirá pontos a melhorar no website. As vantagens são inúmeras. Aproveite este serviço para construir o seu negócio de forma bem sucedida. Reuna informação e conheça melhor os seus utilizadores.
